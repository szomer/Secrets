# Secrets App

- **NodeJS** runtime environment
- **EJS** view engine
- **Bootstrap** CSS framework
- **SCSS** styling syntax
- **MongoDB** with **Mongoose**
- **Passport-local-mongoose** for generating salt/hash fields in db
- **Express Session** middleware
- **Passport** with **Passport-local** strategy
